## seaborn 라이브러리

**seaborn** 은 matplotlib을 더 활용할 수 있는 모듈입니다.

## 소스코드로 알아보기

### import

왜인지는 모르겠으나 `sns`라고 alias합니다.

``` python
import seaborn as sns
```
